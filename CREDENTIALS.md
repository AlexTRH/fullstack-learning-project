# Тестовые учётные данные

Используй эти данные для регистрации и входа в приложение (проходят валидацию на фронте и бэке).

## Регистрация (Sign up)

| Поле              | Значение        |
|-------------------|-----------------|
| **Full Name**     | `Alex Test`     |
| **Username**      | `alextest`      |
| **Email**         | `alex@example.com` |
| **Password**      | `Password1`     |
| **Confirm Password** | `Password1`  |

**Правила:** username 3–30 символов, только буквы/цифры/`_`. Пароль минимум 8 символов, хотя бы одна заглавная, одна строчная и одна цифра.

---

## Вход (Login)

После регистрации можно войти с теми же данными:

| Поле       | Значение        |
|------------|-----------------|
| **Email**  | `alex@example.com` |
| **Password** | `Password1`   |

---

## Второй аккаунт (для тестов подписок)

Сначала зарегистрируй по форме выше, затем зарегистрируй второй аккаунт с этими данными:

| Поле              | Значение           |
|-------------------|--------------------|
| **Full Name**     | `Jane Doe`         |
| **Username**       | `jane_doe`         |
| **Email**         | `jane@example.com`  |
| **Password**      | `Password1`         |
| **Confirm Password** | `Password1`     |

**Вход (Login):**

| Поле       | Значение           |
|------------|--------------------|
| **Email**  | `jane@example.com` |
| **Password** | `Password1`     |

Так можно тестировать подписки: залогинься как `alex@example.com`, открой профиль `jane_doe` по ссылке или `/users/<id>`, нажми Follow; выйди, зайди как `jane@example.com` — подписка отобразится.

---

Другие примеры паролей: `Test1234`, `MyPass1`.  
Другие username: `john_doe`, `user123`.
